# Security Policy

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report vulnerabilities privately by contacting the maintainers and include:

- A clear description of the issue
- Reproduction steps
- Potential impact
- Suggested mitigation (if known)

We will acknowledge reports as quickly as possible and provide follow-up on remediation status.

## Scope Notes

This repository depends on external frameworks (Ray, RLlib, PyTorch, Gymnasium, etc.).  
When reporting a vulnerability, please note whether it is in this project code or an upstream dependency.
